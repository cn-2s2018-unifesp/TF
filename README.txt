Para utilizar o código é necessário instalar a extensão BitVector do pip --- "pip install BitVector"
Para rodar o caso teste é necessário executar o arquivo "run.py" com o python versão 3.X presente nessa pasta
Para trocar a frase criptografada você pode abrir o arquivo "run.py" e trocar a string atribuida a variável "b"
Para trocar a chave utilizada no código você pode, no mesmo arquivo, trocar a string atribuida a variável "a"

O arquivo "criptografador.py" é referente a toda implementação feita pelos integrantes do grupo - Não mecher
